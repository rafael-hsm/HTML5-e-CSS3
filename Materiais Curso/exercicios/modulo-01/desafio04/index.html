Meus emojis favoritos

Durante o curso aprendemos a pesquisar e adicionar emojis atráves da biblioteca do emojipedia, porém no vídeo no youtube um gafanhoto deu a dica que basta teclar logotipo do windows + "." no VS que conseguiremos adicionar emojis.
Farei como o desafio pediu, porém essa outra forma é muito mais prática.

Carinhas

  • 🤣 Código U+1F923
  • 😇 Código U+1F607
  • 😎 Código U+1F60E

Mãos

  • ✌ Código U+270C
  • 🤞 Código U+1F91E
  • 👏 Código U+1F44F

Animais

  • 🐶 Código U+1F436
  • 🐎 Código U+1F40E
  • 🐷 Código U+1F437

Esportes

  • ⚽ Código U+26BD
  • 🏇 Código U+1F3C7
  • 🏀 Código U+1F3C0

Comidas

  • 🍌 Código U+1F34C
  • 🍔 Código U+1F354
  • 🥓 Código U+1F953

Objetos

  • 💌 Código U+1F48C
  • 🔪 Código U+1F52A
  • 🎉 Código U+1F389
